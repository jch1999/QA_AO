## 유용한 UE5 콘솔 명령 (QA 보조)

```
showdebug abilitysystem       -- GAS 어빌리티 상태 시각화
showdebug replication         -- 리플리케이션 그래프 확인
stat net                      -- 네트워크 통계 (패킷 손실/레이턴시)
Net PktLag=200                -- 레이턴시 시뮬레이션 (ms)
Net PktLoss=10                -- 패킷 손실률 시뮬레이션 (%)
log LogAO Verbose             -- 프로젝트 로그 상세 출력
log LogKH Verbose             -- 팀원별 로그 카테고리 활성화
```
